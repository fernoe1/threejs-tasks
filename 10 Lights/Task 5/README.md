Create universal MeshStandardMaterial.



Use that material to create cube, sphere, torus, and plane meshes.



Animate cube, sphere, torus to continuously rotate on both x and y axises.



Create rectArea light.



Using lil-gui, add intensity, width, height sliders to rectArea light.



You must have CSS that makes your canvas take whole screen.



You must have resize event listener that listens to resizes.



You must have 'h' keypress event listener that hides/shows the GUI.
